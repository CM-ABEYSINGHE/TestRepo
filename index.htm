How to use Git Repository

Subversion is an open source version control system. Subversion is developed as a project of the Apache Software Foundation. Below is an simple guide to setup svn in your Ubuntu environment.$ apt-get install subversion$ apt-get install libapache2-svn After downloading packages type $ svn -h to see whether you have configured subversion properly.
